El siguiente proyecto puede ser ejecutado, a traves de los siguientes pasos:
1. Clonar el repositorio de github alojado de manera publica en el link: gh repo clone Brayan2729/Descendiente-Capicua
2. Luego de clonar el repositorio, se puede editar por medio de un entorno de desarrollo visual studio code
3. Teniendo abierto el proyecto en el entorno de desarrollo, lo puede ejecutar y depurar sin contratiempos
